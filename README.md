# Tasks API

A lightweight REST-API to handle task manager clients.

##### Running the project locally

Install dependencies
[For Mac's M1 only] Run `bundle config set force_ruby_platform true` before bundle.

```
bundle
```

Initialize the DB

```
rails db:create
```

```
rails db:migrate
```

Run the server

```
rails s
```
